Welcome to IPSS 🏭
The Intelligent Production Scheduling System — a factory floor simulator that proves AI scheduling beats traditional methods using 4 algorithms, real chaos events, and ML prediction.
🎲
1. Choose Input Mode
Use Random to auto-generate jobs, or Manual to enter specific jobs with your own weights, complexity, and priorities.
🤖
2. Pick Algorithms
Select which of the 4 scheduling algorithms to compare. All selected ones run simultaneously on the same jobs so you can compare results fairly.
⚡
3. Enable Chaos (Optional)
Toggle Chaos Mode to simulate machine breakdowns, material shortages, or power outages. All algorithms face identical disruptions.
▶
4. Run & Explore
Click Run Simulation. Explore 7 tabs: Overview, Gantt Chart, Analytics, History, Job Schedule, Chaos Events, and the Feature Guide.
❓
Help Buttons (?) Everywhere
Every sidebar control has a ? button. Click it to see exactly what that control does, what problem it solves, and how it helps.
💡
Feature Guide Tab
The last tab (💡 Guide) explains every feature in plain language with a clear Problem and Solution for each one.
